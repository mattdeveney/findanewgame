User-based Collaborative Filtering Model

1. Aggregate Gaming Data

We scrape data from the rating aggregator Metacritic. In our model, videogames are our "items to be recommended" and critic publications (e.g. GameSpot, XGN, etc.) are our existing "users." Each publication's set of reviewed games serves as a proxy for that publication's consumption history.

2. Create Your Preference Vector

Next, we figure out which games you've played and how much you like them. This constructs a "profile" of your preferences that we can compare to other users' consumption histories to generate suggestions.

3. Suggest Games that Similar Users Like

Finally, we create a similarity matrix (using the cosine similarity function) between you and other users. Using the "similarity scores" we generate, we figure out which games users that are similar to you have rated highly. We suggest the games with the highest aggregate similarity scores.
